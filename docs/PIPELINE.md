## Database Location

File: C:\VScode\meme-radar\meme_radar.db
Size: ~1.3 MB
Type: SQLite

To view in DB Browser:
1. Open DB Browser for SQLite
2. File > Open Database
3. Navigate to: C:\VScode\meme-radar\meme_radar.db
4. Click "Browse Data" tab
5. Select table (hot_videos, watchlist, etc.)


## Quick Commands

```bash
# Start continuous monitoring (5 min intervals)
python radar.py run -i 5

# Check detection status
python radar.py lowkey status

# View top creators
python radar.py lowkey top

# Manual detection run
python radar.py lowkey run

# Collect TikTok data only
python radar.py collect --platform tiktok
```


## Configuration

File: config/config.yaml

Key settings:
```yaml
tiktok:
  trending_limit: 50     # How many trending videos to fetch
  users: [...]           # Specific accounts to monitor
  hashtags: [...]        # Hashtags to scrape

lowkey_detection:
  min_likes: 100000      # Minimum likes to qualify
  max_followers: 1000000 # Skip accounts above this
  min_spike_factor: 3.0  # Must be 3x their average
```
